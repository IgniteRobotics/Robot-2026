# Spin Move Fix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the broken fixed-field-pivot orbit logic in `spinMoveCommand` with a pure-rotation + calibrated-auto-lateral-velocity approach that correctly moves the robot one lane (1.04m) laterally over a full 360° spin.

**Architecture:** The robot spins at a fixed angular rate while a constant lateral velocity — derived from the target displacement and angular rate — automatically moves it one lane to the driver's left or right. Driver vx is honored; driver vy is added on top of the auto-lateral. Rotation direction and lateral direction are both determined by trigger side × alliance sign, with no dependency on travel direction.

**Tech Stack:** Java 17, WPILib 2026, CTRE Phoenix 6 swerve, Gradle

### Task 1: Rewrite `spinMoveCommand`

**Files:**
- Modify: `src/main/java/frc/robot/subsystems/drive/DrivetrainSubsystem.java` (method `spinMoveCommand` at line 277)

**Background:**
- `pivotDist = robotCenterToEdge + SPIN_MOVE_PIVOT_CLEARANCE_METERS = 0.52m`
- `targetLateral = 2 × pivotDist = 1.04m`
- `autoVy = targetLateral × ω / (2π)` — calibrated so exactly 1.04m of lateral displacement accumulates over one full rotation
- `allianceSign = +1 blue / -1 red`
- `lateralDir = (pivotLeft ? +1 : -1) × allianceSign` — field-Y direction of auto-lateral
- `omegaSign = (pivotLeft ? -1 : +1) × allianceSign` — CW from driver's perspective for left trigger

- [ ] **Step 1: Replace `spinMoveCommand` with the new implementation**

Replace the entire method body (lines 277–358 in `DrivetrainSubsystem.java`) with:

```java
public Command spinMoveCommand(
    boolean pivotLeft, DoubleSupplier vxSupplier, DoubleSupplier vySupplier) {
  SwerveRequest.FieldCentric spinRequest =
      new SwerveRequest.FieldCentric()
          .withDriveRequestType(DriveRequestType.OpenLoopVoltage)
          .withDeadband(DriveConstants.MAX_DRIVE_SPEED * DriveConstants.DEADBAND_FACTOR);

  var state =
      new Object() {
        double startY;
        double targetLateral;
        double autoVy;
        double omegaSign;
        double lateralDir;
        double headingDelta;
        Rotation2d lastHeading;
      };

  return Commands.sequence(
      Commands.runOnce(
          () -> {
            double allianceSign =
                AllianceState.getInstance().getAlliance().equals(Alliance.Blue) ? 1.0 : -1.0;

            state.lateralDir = (pivotLeft ? 1.0 : -1.0) * allianceSign;
            state.omegaSign = (pivotLeft ? -1.0 : 1.0) * allianceSign;

            double pivotDist =
                Constants.RobotConstants.robotCenterToEdge.in(Meters)
                    + DriveConstants.SPIN_MOVE_PIVOT_CLEARANCE_METERS;
            state.targetLateral = 2.0 * pivotDist;

            double omega = DrivePreferences.spinMoveAngularSpeed.getValue();
            state.autoVy = state.targetLateral * omega / (2.0 * Math.PI);

            state.startY = driveState.getCurrentDriveStats().Pose.getY();
            state.lastHeading = driveState.getCurrentDriveStats().Pose.getRotation();
            state.headingDelta = 0.0;
          }),
      Commands.run(
              () -> {
                Rotation2d currentHeading =
                    driveState.getCurrentDriveStats().Pose.getRotation();
                state.headingDelta +=
                    Math.abs(currentHeading.minus(state.lastHeading).getRadians());
                state.lastHeading = currentHeading;

                this.setControl(
                    spinRequest
                        .withVelocityX(vxSupplier.getAsDouble())
                        .withVelocityY(
                            state.autoVy * state.lateralDir + vySupplier.getAsDouble())
                        .withRotationalRate(
                            state.omegaSign
                                * DrivePreferences.spinMoveAngularSpeed.getValue()));
              },
              this)
          .until(
              () -> {
                double currentY = driveState.getCurrentDriveStats().Pose.getY();
                return state.headingDelta >= 2 * Math.PI
                    && Math.abs(currentY - state.startY) >= state.targetLateral;
              }));
}
```
